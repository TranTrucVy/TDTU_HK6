## Hướng dẫn chạy test
1. Cài đặt các thư viện cần thiết:
- Robot Framework
- Selenium Library

2. Cách chạy test: 
Cách 1: Dùng robot IDE
- Mở file robot 
- Chọn các test case cần chạy và nhấn run
Cách 2: Dùng terminal
- Mở terminal (VSCode hoặc phần mềm khác) 
- Di chuyển đến thư mục chứa file robot
- Gõ lệnh: robot <tên file robot>.robot (ví dụ: robot test.robot)
